# VRC Worlds Manager V2 - Custom Edition v1.3.0-2

このリリースは、本家の VRC Worlds Manager V2 をベースに、独自の拡張機能と改善を加えたカスタムバージョンです。

## ✨ 新機能 (New Features)
- **ネイティブエクスポート機能**: 設定画面から、現在のワールド・フォルダ情報をJSON形式（`worlds.json`, `folders.json`）で直接書き出せるようになりました。
- **Chrome拡張機能**: ブラウザで選択したテキストを右クリックしてアプリ内で検索したり、VRChat公式サイトのリンク連携を強化するChrome拡張機能を含めました。
  - ※ リリースに添付されている `chrome-extension-v1.3.0-2.zip` を解凍し、Chromeの拡張機能管理画面（デベロッパーモード）から読み込んでください。

## 🎨 変更点 (Changes)
- **Aboutページのリニューアル**: 開発者情報やサポーター表示を見やすく整理・デザイン変更しました。
- **自動アップデート対応**: このカスタムリポジトリ (`maruo29/VRC-Worlds-Manager-v2-Custom`) から直接アップデートを受信・適用できるようになりました。
- **アプリアイコン**: カスタム版独自のアイコン設定（もし変更していれば）やタイトル表記を "Custom Edition" に変更しました。

## 🐛 修正 (Fixes)
- 特定の環境でバージョンの不整合により発生していたビルドエラーを修正しました。
- ディープリンク（`vrc-worlds-manager://`）の処理を改善しました。

---
**インストール方法**: 下記の Assets から `vrc-worlds-manager_1.3.0-2_x64-setup.exe` をダウンロードして実行してください。
